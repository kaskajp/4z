Local LLM Terminal Assistant

2023-11-02

Overview

A lightweight terminal-based assistant that runs entirely locally using smaller language models. It would help with shell commands, provide context-aware suggestions, and assist with common development tasks - all without sending data to external APIs.

Potential Implementation

  • Use models like Llama 3 8B or Mistral 7B that can run on consumer hardware
  • Implement a shell integration similar to GitHub Copilot CLI
  • Provide context awareness by indexing local project files
  • Create a simple chat interface within the terminal

Progress

  • Basic prototype running with Ollama
  • Working on integrating with command history for better context
