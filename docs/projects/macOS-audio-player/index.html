macOS Audio Player

A minimalist, native audio player for macOS built with Swift and SwiftUI. This project focuses on providing a clean, efficient audio experience with a beautiful interface.

Features

  • Native macOS experience with SwiftUI
  • Gapless playback for continuous listening
  • Minimal UI that stays out of your way
  • Support for all common audio formats
  • Smart playlists and library management
  • Low resource usage

Status

This project is currently in active development. I'm working on the core audio engine and UI components. A public beta is expected in the coming months.

Tech Stack

  • Swift 5.5+
  • SwiftUI for the interface
  • AVFoundation for audio processing
  • Core Data for library management
  • Combine for reactive programming
